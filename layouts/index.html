Buku Catatan

Pemisahan buku catatan keuangan setiap kegiatan. Buku operasional harian masjid, buku catatan infaq buka puasa Senin/Kamis, buku catatan kegiatan qurban, dan kegiatan lainnya.

Input Transaksi Kas

Input pemasukan dan pengeluaran dengan memilih kategori/kelompok transaksi. Ini mempermudah bendahara masjid/mushalla untuk input data transaksi keuangan.

Publikasi Laporan Kas

Dengan sistem ini, laporan keuangan yang sifatnya publik dapat ditampilkan di halaman web. Jamaah/masyarakat dapat melihat laporan kas masjid/mushalla dari sini.

Laporan Kas Bulanan

Laporan posisi kas dalam satu bulan: Saldo awal, pemasukan, pengeluaran, dan saldo akhir bisa dilihat di web masjid/mushalla. Termasuk laporan pada bulan-bulan sebelumnya.

Laporan Kas per Kategori

Masyarakat dapat melihat pemasukan dan pengeluaran kas sesuai kategori/kelompok. Sehingga bisa tahu, laporan kotak infaq berapa, tagihan listrik dan air berapa.

Laporan Kas per Pekan

Lebih detail, masyarakat dapat melihat rincian transaksi per pekan: Transaksi hari/tanggal berapa, pemasukan dari mana, pengeluaran untuk apa, sisa saldo berapa.

Input Jadwal Pengajian

Mempermudah pengurus masjid/mushalla untuk mengelola jadwal pengajian rutin. Bisa input dan edit jadwal, atau dibuat libur.

Publikasi Jadwal Pengajian

Jamaah dapat melihat jadwal pengajian "Hari Ini", "Besok", "Pekan Ini" dan "Pekan Depan". Termasuk "Jadwal Khatib" shalat Jumat.
